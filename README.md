# 🌿 MindMate

**AI-powered Mental Health & Productivity Bot for Students**

---

##  Features

*  Sentiment-aware chatbot (positive / neutral / negative responses)
*  Daily mood & productivity check-ins
*  Simple analytics dashboard (trends & insights)
*  Pomodoro timer for focus sessions
*  Daily motivational quotes

---

## 🛠 Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **NLP:** Natural (basic sentiment analysis)
* **Charts:** Chart.js

---

##  Project Structure

```
mindmate/
├── backend/
├── frontend/
└── README.md
```

---

## 🚀 How to Run

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
